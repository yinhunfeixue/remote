show()method 
public static function show(text:String, title:String, buttonValues:Array = null, closeHandler:Function = null, buttonLabels:Array = null, modal:Boolean = true):Alert

弹出 Alert 控件的静态方法

Parameters

text:String — 在 Alert 控件中显示的文本字符串。
 
title:String — 标题栏中显示的文本字符串
 
buttonValues:Array (default = null) — Alert 控件中放置的按钮的值，可以使用 Alert.OK、Alert.CANCEL、Alert.YES 和 Alert.NO,也可以自定义值 默认是Alert.OK
 
closeHandler:Function (default = null) — 按下 Alert 控件上的任意按钮时将调用的事件处理函数。传递给此处理函数的事件对象是 CloseEvent 的一个实例；此对象的 detail 属性包含按钮 buttonValues中对应的值
 
buttonLabels:Array (default = null) — 按钮上显示的文字，和buttonValues一一对应，若buttonValues中未使用自定义值，此参数可以使用null
 
modal:Boolean (default = true) — 是否以模式方式弹出

Returns
Alert — 弹出的Alert
